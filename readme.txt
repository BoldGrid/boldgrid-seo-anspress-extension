== Description ==

BoldGrid SEO Anspress Extension allows for compatibility of the Anspress plugin and BoldGrid SEO.

== Requirements ==

* PHP 5.3 or higher.

== Installation ==

1. Upload boldgrid-seo-anspress-extension.php to the /wp-content/plugins/ directory.

2. Activate the plugin through the Plugins menu in WordPress.
